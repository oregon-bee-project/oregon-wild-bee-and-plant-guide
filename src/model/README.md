# Executing search_by_location
0. (Once) Set Up Virtual Environment using command ```python -m venv ../../.venv```
1. Activate virtual environment with command ```../../.venv/scripts/activate```
2. Install dependencies with command ```pip install < ../requirements.txt```
3. Start prototype_main with ```python prototype_main.py```
4. On a different tab/terminal start search_by_location with ```python search_by_loaction.py```
5. Output should be printed from the prototype_main terminal
6. Enter 'e' in prototype_main terminal to end process, hit enter to run again
